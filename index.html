Ditulis oleh Daniswara Zavier

Halo teman - teman! Perkenalkan nama saya Daniswara Zavier Putra Akmal atau biasa di panggil Danis. Saya adalah seorang siswa yang berumur 17 tahun ketika website ini dibuat. Saya menempuh pendidikan saya dalam bidang Komputer ini di SMK NEGERI 8 Semarang, dan si website ini saya akan menjelaskan sedikit tentang diri saya

Ketika saya kecil, saya sangat terobsesi dengan dunia penerbangan dan ingin menjadi pilot, saya pun fokus untuk memperdalam bidang akademis seperti pelajaran matematika. Namun, impian itu harus saya kubur karena kebiasaan buruk saya yang banyak bermain video gane sejak kecil. Rasa sedih dan kecewa saya rasakan setelah pemeriksaan kesehatan mata, namun orang tua saya memberi semangat pada saya dan mengatakan bahwa pekerjaan di dunia ini tidak hanya di dunia penerbangan. Ketika saya lulus dari bangku Sekolah Menengah Pertama dengan index nilai 34,19 saya cukup kesulitan untuk mencari Sekolah Menengah Atas. Singkat cerita, teman satu kelas saya saat di Sekolah Menengah Pertama merekomendasikan saya untuk masuk ke SMK NEGERI 8 Semarang dengan jurusan RPL (Rekayasa Perangkat Lunak).
